## 明确排除

- 不重写技能伤害、MP、冷却、学习成本、等级上限或五槽解析规则。
- 不扩展角色技能内容，不补造原版不存在的技能图标或按钮。
- 不顺带原生化宠物、法宝、背包或其他页面；它们由 PG-007 后续独立任务处理。
- 不修改或重新生成 `local-resources/regima/legacy-extraction/` 原始提取结果。
- 原版确实缺少现代可访问交互时，只允许最小现代等价，并必须标记、解释和单独验收。
