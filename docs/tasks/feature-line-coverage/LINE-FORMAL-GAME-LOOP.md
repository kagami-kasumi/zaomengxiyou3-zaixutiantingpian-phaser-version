# LINE-FORMAL-GAME-LOOP 覆盖台账

本文是 `LINE-FORMAL-GAME-LOOP` 的权威覆盖入口。该线由用户在 Stage 1 三关完成后明确前置，目标不是一次性美化全部页面，而是先建立可通关、可观察、可保存、可选关的正式玩家主循环，再逐步闭合完整功能 UI。

## 当前范围

- 玩家可见目标：启动游戏后可选择存档，从天庭地图进入已解锁关卡，在清晰 HUD 与攻击反馈下完成 Stage 1 三关，并在返回地图后保留进度。
- 战斗目标：定位“过于容易死亡”的具体来源；依据原版证据与现代可玩性验证校准玩家/怪物数值、攻击前摇、命中/受击反馈、保护和续航，不凭感觉改常量。
- UI 目标：先完成 HP/MP/经验/等级、技能与重要敌人状态等核心 HUD，再闭合背包、装备、宠物、心法/技能和法宝入口的正式 UI。
- 流程目标：正式启动/存档槽 → 天庭地图 → Stage 1-1/1-2/1-3 → 失败/胜利/退出 → 地图与存档持久化。
- Stage 2-1 内容、更多关卡、网络、支付/活动和与本流程无关的全量内容明确排除。
